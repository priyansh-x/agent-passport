# Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Our Standards

**Do:** Be respectful, constructive, and collaborative. Accept feedback gracefully. Focus on what's best for the project.

**Don't:** Harass, troll, or discriminate. Use sexualized language. Publish others' private information. Engage in conduct that would be inappropriate in a professional setting.

## Enforcement

Violations can be reported to the maintainers via GitHub issues or email. All complaints will be reviewed. Maintainers have the right to remove, edit, or reject contributions that violate this code.
